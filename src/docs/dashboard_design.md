# AI技术生态仪表盘设计文档

## 1. 仪表盘概述

### 1.1 功能目标
- 展示区域AI产业发展态势
- 提供企业数据的可视化分析
- 支持多时间维度的数据查看
- 为决策提供数据支持

### 1.2 核心指标
- 企业总数及增长趋势
- AI应用企业数量及占比
- 百度AI（飞桨/文心）覆盖情况
- 深度合作企业统计

## 2. 仪表盘主要内容

### 2.1 总览指标卡片
- **企业总数**: 显示当前系统中管理的所有企业数量
- **AI采用企业**: 已应用AI技术的企业数量及占比
- **百度AI覆盖**: 使用百度AI服务的企业数量及占比
- **新增企业**: 过去选定时间段内新增的企业数量

### 2.2 分布分析图表
- **按行业分布**: 柱状图展示不同行业AI应用情况
- **按AI场景分布**: 柱状图展示AI应用场景分布
- **AI成熟度分布**: 饼图展示企业AI应用成熟度等级分布
- **按地区分布**: 地图或柱状图展示区域AI发展情况

### 2.3 趋势分析图表
- **时间趋势**: 折线图展示企业增长和AI应用趋势
- **对比分析**: 不同时间段的数据对比
- **预测趋势**: 基于历史数据的未来趋势预测

### 2.4 重点企业展示
- **高优先级企业**: P0优先级企业列表
- **最新加入企业**: 最近加入的AI企业
- **深度合作企业**: 与百度AI深度合作的企业

## 3. 时间维度选择功能

### 3.1 时间跨度选项
- **一周**: 最近7天的数据
- **两周**: 最近14天的数据  
- **一个月**: 最近30天的数据
- **三个月**: 最近90天的数据
- **半年**: 最近180天的数据
- **一年**: 最近365天的数据
- **两年**: 最近730天的数据
- **自定义**: 用户可自定义时间范围

### 3.2 时间维度交互设计
- **顶部时间选择器**: 在仪表盘顶部放置时间选择下拉菜单
- **快速切换按钮**: 提供常用时间跨度的快速切换按钮
- **时间范围滑块**: 支持拖拽选择自定义时间范围
- **相对时间**: 支持"当前季度"、"当前年度"等相对时间选择

### 3.3 时间维度的数据显示
- **数据聚合**: 根据选择的时间段自动聚合数据
- **趋势对比**: 显示当前时间段与上一时间段的对比
- **周期性分析**: 支持同比、环比分析
- **动态刷新**: 时间选择变更时自动刷新所有图表

## 4. 交互功能

### 4.1 图表交互
- **钻取功能**: 点击图表元素可钻取到详细数据
- **悬停提示**: 鼠标悬停显示详细数值和信息
- **选择高亮**: 选择图表元素高亮相关数据

### 4.2 数据筛选
- **行业筛选**: 按行业筛选显示特定行业数据
- **地区筛选**: 按地区筛选显示特定区域数据
- **AI成熟度筛选**: 按AI应用成熟度筛选
- **AI产品筛选**: 按使用的百度AI产品筛选

## 6. 数据更新机制

### 6.1 自动更新
- **定时刷新**: 设定时间间隔自动更新数据
- **实时监听**: 监听数据变更事件实时更新
- **手动刷新**: 提供手动刷新按钮

### 6.2 数据缓存
- **数据缓存**: 缓存历史查询结果提升性能
- **增量更新**: 只更新变化的数据部分
- **预加载**: 预加载用户可能查看的数据

## 7. 响应式设计

### 7.1 不同屏幕适配
- **桌面端**: 全功能显示，多列布局
- **平板端**: 适当简化，双列布局
- **移动端**: 简化功能，单列布局

### 7.2 交互优化
- **触摸友好**: 移动端优化触摸交互
- **加载优化**: 移动端优化数据加载策略
- **性能优先**: 根据设备性能调整动画效果

## 8. 可访问性和国际化

### 8.1 可访问性
- **键盘导航**: 支持键盘操作所有功能
- **屏幕阅读器**: 支持屏幕阅读器访问
- **颜色对比**: 符合可访问性颜色对比度要求

### 8.2 国际化
- **多语言**: 支持中英文界面切换
- **数字格式**: 根据地区设置格式化数字
- **时间格式**: 根据地区设置格式化时间

## 9. 技术实现要求

### 9.1 前端技术
- **图表库**: 使用ECharts或D3.js实现可视化
- **框架**: React + Ant Design或类似组合
- **样式**: neubrutalism设计风格

### 9.2 性能要求
- **加载时间**: 首次加载<3秒
- **响应时间**: 交互响应<0.5秒
- **更新频率**: 支持实时或准实时更新
