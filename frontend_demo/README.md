# 区域AI产业企业数据管理平台 - 前端演示

## 项目概述

这是一个区域AI产业企业数据管理平台的前端演示应用，旨在展示平台的主要功能和用户体验。该平台面向区域产业分析师和运营人员，统一管理AI企业的结构化数据。

## 主要功能

- **仪表板**: 企业总数、AI应用企业、百度AI使用情况、高潜力企业等KPI展示
- **企业管理**: 企业数据的查看、搜索、筛选和编辑
- **企业详情**: 企业基本信息、融资信息、AI应用、百度AI使用情况等详细信息
- **导入导出**: 批量导入企业数据和导出当前数据的功能
- **任务管理**: 企业管理相关的任务和待办事项
- **系统设置**: 个人资料、账户设置、通知设置、系统配置等

## 技术栈

- **框架**: React 18 + TypeScript
- **状态管理**: React Hooks
- **路由**: React Router v6
- **UI组件**: Tailwind CSS + 原生组件
- **数据可视化**: Recharts
- **图标**: Lucide React
- **构建工具**: Vite

## 本地运行

1. 确保已安装 Node.js (v16+) 和 npm
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 在浏览器中访问 `http://localhost:3000`

## 项目结构

src/
