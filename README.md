# Regional AI Industry Enterprise Data Management Platform

A comprehensive data management platform designed for tracking and analyzing AI industry enterprises in regional markets. This platform provides tools for managing enterprise information, tracking AI usage, monitoring implementation stages, and visualizing key metrics.

## Features

- **Enterprise Management**: Track detailed information about AI industry companies
- **AI Application Tracking**: Monitor AI implementations and use cases across enterprises
- **Baidu AI Integration**: Track usage levels of Baidu's AI services
- **Operational Tags**: Categorize enterprises with custom tags for better organization
- **Dashboard Analytics**: Visualize enterprise data with interactive charts and metrics
- **Import/Export**: Import and export data in various formats for integration with other systems
- **Task Management**: Track tasks related to enterprise data management
- **Neubrutalism UI Design**: Unique and visually appealing user interface

## Technologies Used

- **React**: Frontend library for building user interfaces
- **TypeScript**: Type-safe JavaScript development
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Vite**: Fast build tool and development server
- **Recharts**: Charting library for data visualization
- **Lucide React**: Icon library
- **Radix UI**: Accessible UI components

## Installation

1. Clone the repository:
```bash
git clone https://github.com/view2future/enterprise_ai_workstation.git
```

2. Navigate to the frontend directory:
```bash
cd enterprise_ai_workstation/frontend_demo
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.
