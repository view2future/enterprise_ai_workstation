# 区域AI产业企业数据管理平台PRD分析

## 概述
本系统面向区域产业分析师和运营人员，统一管理数百到上千家企业的结构化数据，包括工商基础信息、规模、融资、AI应用场景及百度AI使用情况。

## 目标
- 将现有Excel作为主业务工具的使用频次降低80%以上
- 批量更新500家企业的平均耗时 ≤ 30分钟（含导入校验）
- 标准运营名单生成时间从"人工数小时"降低到 < 10分钟
- 支撑 ≥ 10,000 家企业数据，保持核心列表查询和Dashboard加载时间在可接受范围内

## 核心功能需求
- 数据管理模块：企业数据模型定义，批量导入/更新，单条编辑，变更日志
- 查询与筛选模块：基础搜索，高级筛选，筛选模板
- Dashboard可视化模块：总览卡片，分布图，时间维度趋势
- 数据导出模块：列自定义，条件继承，导出格式支持
- 用户与权限模块：用户管理，角色配置
- 字段与配置模块：字段配置，枚举维护

## 用户体验
- 首次用户体验：引导页，导入模板下载
- 核心体验：从Dashboard进入列表或筛选，企业列表与详情编辑，保存筛选模板与导出
- 高级功能：导入文件大小限制，字段缺失处理，去重冲突处理

## 技术考虑
- 前后端分离架构
- 支持复杂表格、过滤器与图表交互
- 数据库为主，合理设计企业主表、AI应用信息表等
- 支持外部系统集成预留接口
- 性能优化：索引、物化视图、缓存加速
