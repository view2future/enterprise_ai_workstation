# 区域AI产业企业数据管理平台 - 主要功能列表

## 1. 核心企业管理功能
- **企业CRUD操作**: 完整的创建、读取、更新和删除企业记录功能
- **企业详情视图**: 带有基本信息、融资、AI应用和运营数据标签的综合视图
- **业务信息管理**: 存储公司名称、注册信息、法定代表人、资本等核心业务详情
- **融资信息跟踪**: 管理融资轮次、金额、日期和投资者信息
- **AI应用管理**: 跟踪AI用例、实施阶段和ROI指标
- **百度AI使用跟踪**: 详细记录使用的百度AI产品和采用级别
- **运营标签**: 为企业标记运营状态和备注
- **变更日志**: 自动跟踪所有修改并记录用户和时间戳
- **动态字段支持**: 可扩展架构允许添加自定义字段而无需数据库更改
- **审计跟踪**: 完整记录所有对企业的变更历史

## 2. 导入/导出和同步功能
- **批量导入功能**: 从Excel/CSV文件批量导入企业数据
- **字段映射界面**: 用于映射文件列到系统字段的可视化工具
- **导入验证**: 自动检测错误、重复和数据不一致
- **导入进度跟踪**: 后台处理和进度指标
- **导入错误报告**: 详细的失败报告，包括行特定错误描述
- **导出功能**: 将筛选的企业列表导出为Excel/CSV格式
- **自定义字段导出**: 支持导出动态添加的自定义字段
- **导入模板生成**: 自动生成包含所有当前字段的模板
- **数据同步**: 用于跨环境共享数据的包式导出/导入
- **完整性验证**: 同步包的SHA256哈希验证
- **冲突解决**: 导入期间处理重复记录的多种策略
- **格式支持**: 支持CSV和Excel文件格式

## 3. 仪表板和分析功能
- **企业概览仪表板**: 摘要卡片显示企业总数、AI采用率和百度AI使用情况
- **分布可视化**: 按行业、AI场景和成熟度显示企业分布的图表
- **趋势分析**: 显示过去12个月新增/更新企业趋势的时间图表
- **交互式图表**: 可点击图表根据图表段筛选企业列表
- **KPI跟踪**: AI行业指标的关键绩效指标
- **钻取功能**: 点击可视化以查看基础数据
- **筛选集成**: 与主企业列表筛选器同步的仪表板筛选器
- **响应式设计**: 适应不同屏幕尺寸的仪表板布局
- **深色/浅色主题支持**: 多种外观主题供用户偏好
- **可导出可视化**: 导出仪表板图表和数据的功能

## 4. 用户管理和权限功能
- **基于角色的访问控制**: 三种不同权限的用户角色（分析师、运营人员、管理员）
- **用户账户管理**: 创建、激活、停用和密码重置功能
- **权限管理**: 基于用户角色的不同操作粒度权限
- **认证系统**: 安全的JWT认证和刷新令牌
- **会话管理**: 适当的会话处理和安全
- **活动记录**: 用户操作的跟踪用于审计
- **密码安全**: 强密码要求和安全存储
- **账户恢复**: 通过电子邮件的密码重置功能

## 5. 搜索和筛选功能
- **基本搜索**: 按企业名称、注册代码和位置快速搜索
- **高级筛选**: 按行业、员工数、融资阶段、AI使用等多参数筛选
- **筛选模板**: 保存和重用复杂的筛选组合
- **实时筛选**: 应用筛选器时结果的即时更新
- **列筛选**: 直接从表格列标题筛选
- **保存搜索功能**: 保存和分享常用的搜索条件
- **多参数搜索**: 同时组合多个搜索参数
- **搜索历史**: 跟踪最近搜索以便快速访问

这个综合功能列表涵盖了区域AI产业企业数据管理平台的所有关键功能，旨在满足分析师和运营商高效管理企业数据的需求，同时为未来需求提供灵活性。
